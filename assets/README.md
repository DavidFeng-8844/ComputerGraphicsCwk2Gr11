# Assets 资源文件说明

## 已包含的文件（会被 Git 跟踪）

以下文件已经包含在仓库中，`git clone` 后会自动下载：

- **Shader 文件**：`default.vert`, `default.frag` - 必需，代码运行需要
- **字体文件**：`DroidSansMonoDotted.ttf` - 必需，用于文本渲染
- **材质文件**：`*.mtl` - 小文件，保留作为参考
- **许可证文件**：`licenses.txt` - 资源使用许可说明

## 需要手动添加的大文件（不会被 Git 跟踪）

由于文件较大，以下文件不会被 Git 跟踪。如果需要使用这些示例资源，请从原始来源获取或从队友处复制：

### 模型文件
- `parlahti.obj` (~85 MB) - Parlahti 地形模型
- `landingpad.obj` - Landingpad 模型

### 纹理文件
- `L4343A-4k.jpeg` - 4K 纹理贴图（parlahti.mtl 中引用）

## 如何获取这些文件

1. **从队友处复制**：如果队友有这些文件，直接复制到 `assets/cw2/` 目录即可
2. **从课程资料获取**：查看课程提供的原始资源包
3. **使用自己的资源**：你也可以使用自己的模型和纹理文件

## 注意事项

- 这些大文件是**示例资源**，不是代码运行所必需的
- 你可以使用任何 `.obj` 模型文件和纹理文件
- 如果使用自己的资源，确保文件路径和格式正确
